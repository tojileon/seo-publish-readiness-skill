# Project Learnings

- Treat the Search Intent Map as a default output for SEO audits. Skip it only when the user explicitly asks for technical-only checks or there is no page/content surface to assess.
- For image-alt SEO findings, distinguish informative product-proof images from decorative or duplicate screenshots. Empty alt can be correct for decorative duplicates; report the issue around missing context for important images, not raw empty-alt counts.
